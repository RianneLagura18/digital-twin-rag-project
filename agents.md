# AI Agents Guide

## Tech Stack
- Frontend: React / Next.js
- Backend: Node.js / Express
- Database: (optional, if meron kayo)
- AI: OpenAI API (RAG-based system)

## Project Overview
This project is a Digital Twin system that simulates user interaction using AI. It answers queries based on structured knowledge and documents.

## Architecture
- Frontend handles UI and user input
- Backend processes requests and connects to AI
- AI uses knowledge base from PRD and data files

## Rules for AI Agents
- Always refer to /docs/prd.md for requirements
- Follow project structure strictly
- Do not generate random or unrelated features
- Keep outputs aligned with project goals
